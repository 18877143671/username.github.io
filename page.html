安利几个学习前端的好工具

2017.09.25 进击的码农，有趣的分享 5 °C

这里列出的都是本人所熟悉的工具，本文将不定期持续更新，我每找到好的都可以在这里贴出来。

学习类

MDN：
一个由火狐提供的开发者学习平台，全称 “Mozilla Developer Network”，拥有较全的教程，但部分较新标准只有英文文档。

W3School：
国内较为知名的在线教程平台，提供前端和后端多个语言的教程，但教程使用的方法和标准可能相对老旧。

测试类

CanIUse：
一个提供查询 CSS 属性、JS 方法兼容性的网站。想尽可能兼容老旧浏览器的必备之选。

JSFiddle：
一个在线编辑 HTML+CSS+JS 的平台，不需要注册就可以保存你写的代码（注意地址一旦忘记就找不到了）

Google PageSpeed：
一个谷歌提供的测试工具，可以提供网页加载性能的优化建议。

CSS Tricks：
一个提供前端开发优化教程的网站，不过是英文。

Webkit 浏览器内置 CSS：
想要制作新效果和更好的重置浏览器样式？快来参考一下 Webkit 内置的样式表吧。

框架类

Kico Style：
我制作的 CSS 框架，提供的样式较为简单，方便新手学习架构和布局。还提供了较常用的交互组件，方便调用。

Bootstrap：
恐怕是世界上最多人用的 CSS 框架了，提供各种各样的布局方案，你甚至可以不写 CSS 就可以完成一个前端网页的制作。

JQuery
当下最为主流的 JS 库，基于 JQ 开发的插件更是齐全。有自己的语法

Vue.JS：
挺简单易用的 JS 框架，我还在学习使用中。

软件类

Notepad++
简洁快速的运行在 Windows 平台的编辑器，支持多种编程语言。

YUI Compressor
提供 JS、CSS 代码压缩的一款工具。

WebStorm
来自 JetBrains 的高度集成化 IDE，同时还支持一些 JS 框架的自动补全，并可以构建 node.JS 应用。但要收费

Sublime Text
据说比较精简，本人没用过，是大佬推荐的。但要收费

VS Code
来自微软的免费代码编辑器，支持多种语言的编辑和补全，还可以自己安装扩展插件来获得更多的功能。唯一不理解的是它界面竟然是基于 HTML 构建的？

也欢迎你在这里推荐更多实用的工具，以便让更多的人知道其魅力。
